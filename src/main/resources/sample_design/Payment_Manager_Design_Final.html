‚öôÔ∏è Technical Specifications

üñ•Ô∏è Platform & Performance
  • Target: Windows 7+
  • Hardware: 2-4GB RAM, older CPUs
  • Startup time: <2 seconds
  • Cheque creation: <1 second/step
  • DB queries: <200ms
üîê Security & Storage
  • PIN: SHA-256 hash (not plaintext)
  • Database: SQLite in %APPDATA%
  • Backup/Restore: Built-in
  • Optional: File encryption

üìä Database Tables

  • users - User credentials and info
  • vendors - Vendor master data
  • bank_accounts - Bank account details
  • cheque_books - Cheque series management
  • cheques - All cheques created/printed
  • cheque_layout_templates - Print layout offsets per bank
